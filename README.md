# NAME

protorobotics.org

# DESCRIPTION

Public facing website for PROTO.  The site is (mostly) static, but calls back
to a server when needed for contact forms and the like.  This is so if the
server goes down, the github pages site remains up (it "gracefully degrades").

At the moment the site should be primarily one HTML file and one CSS file, but
will eventually expand into an SSG.
