PROTO Atelier Static Site
========================

This zip is a static HTML/CSS/JS snapshot of the PROTO Atelier theme design.
It is ready for GitHub Pages.

Files
- index.html (homepage)
- products.html, solutions.html, resources.html, pricing.html, impact.html, contact.html, shop.html
- coming-soon.html (optional)
- assets/css/site.css (includes theme vars + minimal block layout + theme CSS)
- assets/js/motion.js (scroll reveal + subtle parallax, motion-safe)

GitHub Pages
1) Create a repo, add these files to the repo root.
2) Repo Settings -> Pages -> Deploy from branch -> main / root.
3) Wait for GitHub to publish.

Notes
- This static version uses a placeholder "order via email" button.
- For full ecommerce, use WooCommerce on WordPress or connect a hosted checkout.
